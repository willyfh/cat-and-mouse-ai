Panduan menggunakan program :
1. Copy seluruh file yang terdapat pada folder bin
2. Pastikan seluruh asset (file image dan text) berada dalam satu folder
3. Gunakan Command Prompt dan jalankan perintah appletviewer SwingApplet.java pada folder yang telah dicopykan tersebut
4. Pastikan JDK telah terinstall pada komputer.
5. Program siap dijalankan.
